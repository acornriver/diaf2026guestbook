방명록 데이터를 실시간으로 구글 스프레드시트에 저장하기 위한 설정입니다. 아래 안내에 따라 Google Apps Script를 배포하고 생성된 **웹앱 URL**을 붙여넣어 주세요.

https://script.google.com/macros/s/... 형태의 URL을 입력하세요. 빈칸으로 두면 데모 모드로 작동합니다.
HOW TO SET UP APPS SCRIPT

연동 방법 가이드

  1. 구글 드라이브에서 새로운 **구글 스프레드시트**를 만듭니다.
  2. 메뉴에서 **확장 프로그램 (Extensions) > Apps Script**를 클릭합니다.
  3. 기존 코드를 모두 지우고, 아래의 소스코드를 복사해서 붙여넣습니다.
  4. 오른쪽 위의 **배포 (Deploy) > 새 배포 (New deployment)**를 클릭합니다.
  5. 유형 선택에서 톱니바퀴를 눌러 **웹앱 (Web app)**을 선택합니다.
  6. 설정을 다음과 같이 지정하고 **배포**를 클릭합니다:
    • 설명: 방명록 API
    • 다음 사용자 인증 정보로 실행: **웹앱을 실행하는 사용자 (나)**
    • 액세스할 수 있는 사용자: **모든 사람 (Anyone)**
  7. 액세스 승인 창이 뜨면 구글 계정으로 로그인한 뒤 권한을 승인합니다.
  8. 생성된 **웹앱 URL**을 복사하여 왼쪽 입력 칸에 저장합니다.
Google Apps Script 소스코드 
function doGet(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  // 시트가 완전히 비어있을 경우 헤더 생성
  if (sheet.getLastRow() === 0) {
    sheet.appendRow(["Timestamp", "Name", "Message", "Vibe"]);
  }
  
  var rows = sheet.getDataRange().getValues();
  var data = [];
  
  if (rows.length > 1) {
    for (var i = 1; i < rows.length; i++) {
      var row = rows[i];
      var dateVal = row[0];
      var formattedDate = "";
      if (dateVal instanceof Date) {
        formattedDate = Utilities.formatDate(dateVal, Session.getScriptTimeZone(), "yyyy-MM-dd HH:mm:ss");
      } else {
        formattedDate = String(dateVal);
      }
      
      data.push({
        timestamp: formattedDate,
        name: String(row[1]),
        message: String(row[2]),
        vibe: String(row[3] || 'ore')
      });
    }
  }
  
  return ContentService.createTextOutput(JSON.stringify(data))
    .setMimeType(ContentService.MimeType.JSON);
}

function doPost(e) {
  try {
    var name = "";
    var message = "";
    var vibe = "ore";
    
    // JSON 또는 Form Parameters 파싱
    if (e.postData && e.postData.contents) {
      try {
        var params = JSON.parse(e.postData.contents);
        name = params.name || "";
        message = params.message || "";
        vibe = params.vibe || "ore";
      } catch (parseError) {
        name = e.parameter.name || "";
        message = e.parameter.message || "";
        vibe = e.parameter.vibe || "ore";
      }
    } else {
      name = e.parameter.name || "";
      message = e.parameter.message || "";
      vibe = e.parameter.vibe || "ore";
    }
    
    if (!name || !message) {
      return ContentService.createTextOutput(JSON.stringify({ result: 'error', error: '이름과 메시지는 필수 항목입니다.' }))
        .setMimeType(ContentService.MimeType.JSON);
    }
    
    var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    // 첫 방문자 자동 헤더 생성
    if (sheet.getLastRow() === 0) {
      sheet.appendRow(["Timestamp", "Name", "Message", "Vibe"]);
    }
    
    var timestamp = new Date();
    sheet.appendRow([timestamp, name, message, vibe]);
    
    return ContentService.createTextOutput(JSON.stringify({ result: 'success' }))
      .setMimeType(ContentService.MimeType.JSON);
  } catch(error) {
    return ContentService.createTextOutput(JSON.stringify({ result: 'error', error: error.toString() }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}
